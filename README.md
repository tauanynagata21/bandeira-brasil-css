# Bandeira do Brasil com CSS

Projeto desenvolvido em sala de aula com o objetivo de praticar CSS e posicionamento de elementos na tela.

## Objetivo
Recriar a bandeira do Brasil utilizando apenas HTML e CSS, sem uso de imagens.

## Conceitos praticados
- Posicionamento com CSS (absolute positioning)
- Uso de transform (rotate)
- Manipulação de cores e formas
- Camadas de elementos (sobreposição)

## Tecnologias utilizadas
- HTML5
- CSS inline (estudo inicial)

## Observação
Projeto desenvolvido durante o aprendizado de CSS para prática de layout e posicionamento de elementos.
